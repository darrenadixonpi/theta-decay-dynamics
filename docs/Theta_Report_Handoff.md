## Build Architecture
The report was built from a single Python script (~1,500 lines) that:
1. Defines all styles, colors, and helper functions
2. Assembles content as a `story` list of reportlab flowables
3. Generates the PDF in one `doc.build(story)` call
4. Charts were generated by separate Python scripts and saved as PNGs in a `charts_v2/` directory
5. The main build script references these PNGs via an `img()` helper function

## Literature Citations Used (with specific findings incorporated)
1. **Emery, Guo & Su (2008)** — Confirmed T* formula analytically; max theta ≠ at strike
2. **Carr & Wu (2009)** — Variance risk premium foundation
3. **Bollerslev, Tauchen & Zhou (2009)** — VRP predicts equity returns
4. **Dew-Becker (2023)** — VRP alpha declining toward zero
5. **Muravyev & Pearson (2020)** — Equity option effective spreads avg ~2.2%
6. **Cao, Jacobs & Ke (2024)** — OTM spreads wider than ATM; widening during stress
7. **Kou (2002)** — Double-exponential jump model; better fit than Merton for asymmetric jumps
8. **Ramezani & Zeng (2007)** — Both Kou and Merton outperform BS empirically
9. **Merton (1976)** — Jump-diffusion model specification
10. **Black & Scholes (1973)** — Core pricing model
11. **tastylive research** — 45 DTE / 50% profit / 21 DTE conventions (attributed as [Convention])

## Simulation Parameters
- Risk-free rate: r = 5%
- Default VRP: IV = 25%, realized vol = 18% (7 percentage point premium)
- Monte Carlo: numpy Mersenne Twister PRNG with fixed seeds for reproducibility
- Path counts: 300–1,500 per analysis depending on section
- Time step: dt = 1/252 (one trading day)
