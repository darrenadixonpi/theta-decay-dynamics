PROJECT CONTEXT

This report investigates whether volatility surface features contain predictive information about forward equity returns.

Key features:
- ATM volatility (level)
- Skew (asymmetry)
- Term structure (slope/inversion)

Workflow:
Vol surface → feature extraction → signal construction → forward return analysis

CRITICAL CONSTRAINTS:

1. This is NOT a trading strategy.
2. Do NOT optimize or propose improvements.
3. Do NOT discuss PnL or performance as a strategy.
4. Focus ONLY on interpretation of signals.

Core finding direction:

- Volatility spikes → associated with positive forward returns (mean reversion)
- Skew spikes → associated with negative forward returns (downside risk)
- Term inversion → weaker negative signal

Limitations:

- Small sample size
- Sparse signal occurrences
- Results are directional, not definitive

Objective of this section:

To explain how volatility surface signals relate to future returns in a clear, structured, and economically interpretable way.
